Typography

The font I will be using or this site is a google font called Source Sans Pro. It is a Sans Serif font. I will be using the Regular weight for the headings and the Extra-Light weight for the main typography.


To keep the pages consistant I will use the font sizes as follows

  • Body/main: 16px
  • h1 phone size: 1.5rem
  • h2 phone size: 1rem
  • h1 tablet/desktop size: 2rem
  • h2 tablet/desktop size: 1.5rem
